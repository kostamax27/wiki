# Паутина
При добыче руды автоматически добываются все соседние блоки того же типа. Зачарование, которое накладывается на кирку, применяется ко всей группе добытых блоков.

**Способы получения:**
- При зачаровывании ботинок
- При зачаровывании книг
- При открытии кейса с книгами
- При покупке в /shop
