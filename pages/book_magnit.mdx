# Магнит
Зачарование накладывается на инструменты, которое позволяет при вскапывании блоков переносить предметы сразу в инвентарь.


**Способы получения:**
- При зачаровывании кирки
- При зачаровывании книг
- При открытии кейса с книгами
- При покупке в /shop


```
Максимальный уровень зачарования — 1
```
