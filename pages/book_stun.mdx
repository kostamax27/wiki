# Ступор

Зачарование для трезубца. При попадании в игрока брошенным трезубцем с определённым шансом накладывается негативный эффект.


**Список негативных эффектов**
- Слабость
- Замедление

**Примечание** : уровни (Дописать)

**Способы получения:**
- При зачаровывании трезубца
- При зачаровывании книг
- При открытии кейса с книгами
- При покупке в /shop

```
Максимальный уровень зачарования — 3
```
