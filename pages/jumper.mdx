## Попрыгун
При ношении предмета с данным зачарованием у вас будет постоянный эффект "Прыгучесть", который работает исключительно на ботинках

**Способы получения:**
- При зачаровывании ботинок
- При зачаровывании книг
- При открытии кейса с книгами
```
Уровень эффекта равен уровню зачарования, максимальный уровень зачарования — 2.
```
