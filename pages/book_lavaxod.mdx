# Лаваход

Зачарование применяется к ботинкам. При наступлении на лаву блок жидкости мгновенно превращается в обсидиан, позволяя добыть его в случае надобности

**Способы получения:**
- При зачаровывании ботинок
- При зачаровывании книг
- При открытии кейса с книгами
- При покупке в /shop

```
Максимальный уровень зачарования — 1.
```
