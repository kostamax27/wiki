## Возвращение
Работает по принципу зачарования "Верность", однако трезубец моментально возвращается в инвентарь с определённым шансом, зачарование накладывается на **трезубец**.

**Способы получения:**
- При зачаровывании трезубца
- При зачаровывании книг
- При открытии кейса с книгами

```
Максимальный уровень зачарования — 1.
```
