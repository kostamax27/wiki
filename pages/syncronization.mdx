# Синхронизация

При переходах между режимами, у пользователя сохраняется информация о том с чем он находился на прошлой анархии, а синхронизируюется следующее :

- Инвентарь
- Баланс монет
- Баланс токенов
- Содержимое эндер-сундука
- Привилегия

## Что не синхронизирует

Анархии синхронизируются между собой, за исключением версий 1.21 и 1.16.5. Перенести вещи с одной анархии на другую нельзя, синхронизируются только баланс токенов и привилегии.

```
Дополнительно! Статистика на определенном режиме, не переносится на другой (часы, убийства, смерти)
```
